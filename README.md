# Instituto Cruzeiro do Sul — Plataforma para ONGs  
**Entrega I – Fundamentos e Estruturação (HTML5)**  

Este projeto foi desenvolvido como parte da disciplina de **Desenvolvimento Front-End**, com foco na aplicação prática dos fundamentos de **HTML5**, **semântica**, **formulários**, **multimídia** e **estruturação profissional** de um site.  

---

## Objetivo Geral  
Desenvolver a base estrutural de uma **plataforma web completa** para Organizações Não Governamentais (ONGs), permitindo divulgar projetos, captar recursos e engajar voluntários.

---

## Estrutura do Projeto  
├── index.html # Página inicial - missão, visão e contato
├── projetos.html # Página de projetos sociais e doações
├── cadastro.html # Página de cadastro de voluntários/doadores
│
├── assets/
│ ├── css/
│ │ └── style.css # Estilos globais e responsividade
│ ├── js/
│ │ └── main.js # Máscaras e interações básicas
│ └── images/
│ └── edu-digital.svg # Imagem exemplo otimizada
│
├── .gitignore
└── README.md

Tecnologias Utilizadas
- **HTML5 Semântico** (estrutura e acessibilidade)  
- **CSS3 Responsivo** (mobile-first e layout flexível)  
- **JavaScript (básico)** para máscaras e validações  
- **Padrões WCAG 2.1 AA** (acessibilidade)  
- **Boas práticas de SEO** (meta tags e estrutura hierárquica)

Páginas Desenvolvidas

| Página | Descrição |
|--------|------------|
| `index.html` | Apresenta a organização, missão, visão, valores e contatos. |
| `projetos.html` | Exibe projetos sociais, formas de doação e progresso de campanhas. |
| `cadastro.html` | Formulário completo para cadastro de voluntários e doadores. |

Requisitos Atendidos
- ✅ Estrutura HTML5 semântica e validada (W3C Validator)  
- ✅ Formulário com validação nativa e agrupamento lógico  
- ✅ Máscaras para CPF, telefone e CEP  
- ✅ Design responsivo (mobile, tablet, desktop)  
- ✅ Acessibilidade (navegação por teclado, contraste, skip link)
